Citation
  https://www.kaggle.com/tamber/steam-video-games/data

Attributes
  This dataset is a list of user behaviors, with columns: user-id, game-title, behavior-name, value. 
  The behaviors included are 'purchase' and 'play'. The value indicates the degree to which the behavior was performed - in the case of 'purchase' the value is always 1, and in the case of 'play' the value represents the number of hours the user has played the game.
  This dataset is generated entirely from public Steam data.

Problem Statement
  To build a recommender system based on user purchases using user-based collaborative filtering.
